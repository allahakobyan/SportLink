# SportLink

SportLink is a mobile fitness and social networking application developed using .NET MAUI and C#.  
The app helps users stay motivated, connect with workout partners, join events, and participate in a community-driven fitness environment.

## Features

- User registration and login
- Email verification system
- Create and join sports events
- Paid and free event system
- Create posts and community polls
- Interactive social feed
- Event details and participant management
- SOS safety feature
- User profiles and community interaction

## Technologies Used

- .NET MAUI
- C#
- XAML
- Visual Studio
- Git & GitHub

## Project Structure
SportLink.Mobile   → Mobile application (UI & app logic)
SportLink.API      → Backend/API project
SportLink.Shared   → Shared models and common classes
